# E-Music Application 

A full-stack music streaming web application built with Django and Python.

## Features
- User authentication and registration
- Music catalog with search functionality
- Play, browse and manage music tracks
- MVC architecture with Django ORM

## Tech Stack
- **Backend:** Python, Django
- **Database:** SQLite3
- **Frontend:** HTML, CSS, Django Templates
- **Auth:** Django Authentication System

## Project Structure
- `accounts/` — user authentication and profiles
- `myproject/` — core Django settings and URLs
- `templates/` — HTML templates

## Run Locally
```bash
git clone https://github.com/cdeekshith11/E_music_application
cd E_music_application/Emusic
pip install django
python manage.py runserver
```
